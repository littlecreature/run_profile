\# RunProfile



!\\\[Logo](addons/run\\\_profile/logo.svg)



RunProfile is a plugin for Godot 4.6 that allows you to change the location of the Editor game window. This plugin a icon in the top right corner of the screen with a popup menu to choose between profiles.



You can edit or add to the profiles, by opening the profiles.tres in the editor and then use the inspector to change any values.



> To prevent RunProfile from activating for everyone when you put your code online, you can simply add "addons/run\\\\\\\_profile" to your project's .gitignore.
